## ğŸ› ï¸ What's New

**Initial v1.0.0 release includes:**

### Core Features
- Multi-platform download support (YouTube, SoundCloud, Spotify, TikTok, Instagram, Vimeo, Twitter, and 1000+ more)
- High-quality audio downloads (up to 320kbps)
- Multiple format support (MP3, M4A, FLAC, Opus, OGG)
- Real-time progress tracking with speed and ETA
- Up to 3 concurrent downloads

### Playlist Features
- Automatic playlist detection
- Individual track progress tracking
- Smart retry for failed items
- Optional separate folders for playlists
- Resume interrupted playlists

### User Interface
- Modern ImGui-based GUI
- Responsive 30 FPS rendering
- Comprehensive settings panel
- Download history with search
- Drag & drop support
- Quick file/folder access

### Advanced
- HTTP/SOCKS5 proxy support
- Service availability checking
- Comprehensive yt-dlp configuration
- Browser cookie integration
- API key support (optional)
- Rate limiting options
- Retry mechanisms
